# Number Guessing Game (Python)

This is a simple number guessing game written in Python.  
The program randomly selects a number between 1 and 100, and the user has to guess it.

Based on the userâ€™s input, the program gives hints until the correct number is guessed.

---

## How the Game Works
1. The computer generates a random number between 1 and 100.
2. The user enters a number as a guess.
3. If the guess is smaller, the program asks to guess a bigger number.
4. If the guess is larger, the program asks to guess a smaller number.
5. The game continues until the correct number is guessed.

---

- Python 3.x
